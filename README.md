## 📌 项目简介

本项目基于通义千问大模型与 LoRA 微调技术，开发一个支持**个性化定制**的旅游攻略推荐系统。通过接入本地数据库并结合用户输入，实现对出行路线、美食推荐、时间安排的全面定制与优化。

- 🔄 实时生成旅游路线规划  
- 🍜 地方特色美食智能推荐  
- 🎯 支持个性偏好与约束条件输入  
- 📈 可视化展示旅行路径与建议
